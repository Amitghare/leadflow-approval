⚙️ One-Time Setup — Configure Make.com URL Params

This page reads leads and the template directly from Google Sheets.
Make.com needs to send the sheet details as URL params. Do this once.

1 Find your Google Sheet ID
Open the sheet. Copy the bold section from the URL:
https://docs.google.com/spreadsheets/d/COPY_THIS_PART/edit
2 Find GID numbers for Leads and Templates tabs
Click the Leads tab → look at the URL:
…#gid=THIS_NUMBER
Repeat for the Templates tab. Note both numbers.
3 Share the sheet publicly
File → Share → Share with others → Change to Anyone with the link → Viewer
Required so the browser can fetch the CSV without a login wall.
4 Required: Leads tab column headers (exact)
Name  Email  Job Title  Company Name  Batch Date  Status
Make.com must write Batch Date as YYYY-MM-DD (ISO format).
5 Required: Templates tab column headers (exact)
Date  Template Body  Status
Make.com writes Claude's template here after Scenario 1 runs.
Date must match YYYY-MM-DD format.
6 Make.com sends this URL in the review email
Use the HTTP module or Set Variable to build this URL, then include it in the email body:
https://YOUR-USERNAME.github.io/approve.html?webhook=WEBHOOK_URL&batch=YYYY-MM-DD&from=amitg@futurismtech.com&sheetId=SHEET_ID&leadsGid=LEADS_GID&templatesGid=TEMPLATES_GID&senderName=Amit+G&meetingLink=YOUR_CALENDLY_URL
